> 版本: v1.0

> 日期: 2026-02-06

## 一、报价原则

### 1.1 基本公式
```
总价 = 基础价格 × 数量 + 工艺附加费 + 设计费（可选）
```

### 1.2 阶梯定价
数量越多，单价越低。每个产品有自己的阶梯规则。

### 1.3 最低起订
- 名片：100张起
- 喷绘：0.5㎡起
- 画册：50本起

## 七、报价函数说明

### 7.1 tenant.calc_card_price()
名片报价函数
- 输入：数量、材质、工艺列表
- 输出：总价、单价、明细

### 7.2 tenant.calc_print_price()
喷绘报价函数
- 输入：宽、高、材质、工艺列表
- 输出：总价、面积、明细

### 7.3 tenant.calc_booklet_price()
画册报价函数
- 输入：数量、规格、页数、纸张、装订方式
- 输出：总价、单价、明细

### 7.4 tenant.calc_craft_fee()
工艺附加费计算
- 输入：工艺类型、面积/数量
- 输出：附加费用

## 八、注意事项

1. **最低消费**：部分产品有最低消费，低于起订量按起订量计价
2. **损耗**：大批量订单预留 3-5% 损耗
3. **加急费**：24小时内 +50%，48小时内 +30%
4. **运费**：市内免费送货（满 ¥200），外地按实际计算
5. **税费**：以上价格不含税，开票 +6%
