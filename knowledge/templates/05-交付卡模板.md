## 📦 交付流程

### 标准流程

```
1. 订单确认
   ├─ 需求确认
   ├─ 文件审核
   └─ 价格确认
   ↓
2. 生产排期
   ├─ 生产计划
   ├─ 材料采购
   └─ 进度跟踪
   ↓
3. 质量检验
   ├─ 首件确认
   ├─ 过程检验
   └─ 成品检验
   ↓
4. 包装发货
   ├─ 成品包装
   ├─ 物流发货
   └─ 跟踪配送
   ↓
5. 客户签收
   ├─ 客户验收
   ├─ 问题反馈
   └─ 售后服务
```

### 时间节点

| 阶段 | 标准时间 | 关键动作 | 责任人 |
|------|---------|---------|--------|
| 订单确认 | 下单后1h | 需求确认+文件审核 | 业务员 |
| 生产排期 | 确认后2h | 排入生产计划 | 生产经理 |
| 首件确认 | 开机后1h | 客户确认样品 | QA |
| 生产完成 | 按约定 | 成品检验 | QA |
| 包装发货 | 完成后4h | 打包+发货 | 仓库 |
| 客户签收 | 发货后1-2天 | 验收确认 | 客户 |

## ✅ 质量检验

### 检验标准

#### 装订类产品（画册/书籍）

| 检验项 | 标准 | 检验方法 | 抽检比例 |
|--------|------|---------|---------|
| **外观质量** | | | |
| 尺寸精度 | ±1mm | 卡尺测量 | 100% |
| 色彩准确度 | ΔE≤3 | 目视/色差仪 | 20% |
| 套印精度 | ±0.3mm | 放大镜 | 20% |
| 表面清洁 | 无污渍/划痕 | 目视 | 100% |
| **装订质量** | | | |
| 装订牢固度 | 180°翻开不脱页 | 手工测试 | 10% |
| 胶线均匀 | 无胶溢出/空白 | 目视 | 20% |
| 书脊平整 | 无起泡/褶皱 | 目视 | 100% |
| **内容质量** | | | |
| 页码顺序 | 100%正确 | 逐页检查 | 20% |
| 内容完整 | 无缺页/漏印 | 逐页检查 | 20% |
| 文字清晰 | 无模糊/断笔 | 目视 | 20% |

#### 喷绘类产品（展架/海报）

| 检验项 | 标准 | 检验方法 | 抽检比例 |
|--------|------|---------|---------|
| 尺寸精度 | ±5mm | 卷尺测量 | 100% |
| 画面质量 | 无色差/条纹 | 目视 | 100% |
| 材质 | 符合要求 | 目视/手感 | 100% |
| 安装配件 | 齐全/完好 | 清点 | 100% |

### 检验流程

**首件确认（必须）**
1. 生产前打样
2. 客户确认样品
3. 签字确认后批量生产

**过程检验（抽检）**
1. 印刷首张检查
2. 生产中期抽检
3. 发现问题立即停机

**成品检验（必须）**
1. 全检外观质量
2. 抽检内在质量
3. 不合格品返工

## 🔄 售后服务

### 质量保证

**质量承诺：**
- 印刷质量问题：全额退换
- 装订质量问题：免费返工
- 设计错误（我方）：免费重做
- 客户文件问题：协商处理

### 售后时效

| 问题类型 | 响应时间 | 处理时间 | 责任方 |
|---------|---------|---------|--------|
| 质量问题 | 2小时 | 24-48小时 | PrintShop |
| 数量错误 | 1小时 | 24小时 | PrintShop |
| 配送延误 | 即时 | 协商 | 物流/PrintShop |
| 客户文件问题 | 1小时 | 客户修改 | 客户 |

### 退换货政策

**可退换情况：**
- 质量问题（我方责任）
- 数量错误
- 产品损坏（运输）

**不可退换情况：**
- 客户文件错误
- 客户主观不满意（非质量问题）
- 定制产品（除质量问题）

**退换流程：**
1. 客户提出问题（电话/微信）
2. 拍照/录像确认问题
3. 确认责任方
4. 协商解决方案
5. 执行退换货
