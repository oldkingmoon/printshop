#### 标准询价流程

```

📏 尺寸：A4还是其他尺寸？

### 3. 价格协商话术

#### 客户砍价

**方案A：解释价值**
```
[客户]总，我理解您对价格的考虑。

我给您算过了，这个价格已经是
很实惠的了：

✅ 我们用的是[品牌]铜版纸，
   不是普通纸
✅ 印刷机是[品牌]，色彩还原度高
✅ 免费设计+3次修改
✅ 市内免费送货

同行可能便宜10-20元，但材质
和工艺差很多，您可以对比看看。

咱们做生意讲的是性价比，
您说对吧？😊
```

**方案B：适度让步**
```
[客户]总，看您是诚心要做，
我跟老板申请一下。

数量是[数量]对吧？
如果确定下单，我给您申请到
¥XX/本，让您XX元。

这真的是底价了，您看行吗？

确定的话我马上安排！🤝
```

**方案C：增加价值**
```
[客户]总，价格确实没法再降了，
但我可以给您增加点服务：

✅ 免费加急（3天变2天）
✅ 多送5本样品
✅ 下次订单优惠10%

您看这样可以吗？😊
```

#### 客户嫌贵

```
我理解您的感受。

确实，市场上有更便宜的，
但咱们的优势是：

1️⃣ 质量保证

   - 装订牢固，不脱页
   - 有问题免费重做

2️⃣ 服务到位
   - 免费设计
   - 送货上门
   - 售后无忧

3️⃣ 交期准时
   - 承诺3天，绝不拖延
   - 加急可做

便宜的可能省100元，
但返工浪费的时间和精力
可能值1000元。

您说对不对？😊

## 🎯 场景应对策略

### 常见异议处理

| 客户异议 | 应对策略 |
|---------|---------|
| "太贵了" | 强调价值+适度让步 |
| "别家更便宜" | 质量对比+服务差异 |
| "能不能便宜点" | 数量优惠+增值服务 |
| "我考虑一下" | 限时优惠+跟进策略 |
| "文件没问题吧" | 专业审核+提前预警 |
| "能不能快点" | 加急服务+额外费用 |

### 客户类型应对

#### 价格敏感型
- **特征：** 主要关注价格
- **策略：** 
  - 提供标准方案
  - 强调性价比
  - 数量优惠
  - 避免过度推销高端方案

#### 质量要求型
- **特征：** 注重品质和细节
- **策略：**
  - 详细介绍材质工艺
  - 提供样品参考
  - 强调质量保证
  - 推荐高端方案

#### 时间紧迫型
- **特征：** 急需快速交付
- **策略：**
  - 快速响应
  - 加急服务
  - 清晰告知时间节点
  - 保证准时交付

#### 犹豫不决型
- **特征：** 迟迟不下决定
- **策略：**
  - 提供多种方案对比
  - 限时优惠促单
  - 客户案例分享
  - 降低决策风险

## 💡 沟通技巧

### 黄金法则

1. **倾听优先**
   - 让客户说完
   - 记录关键信息
   - 确认理解正确

2. **专业表达**
   - 使用专业术语（适度）
   - 避免口语化
   - 逻辑清晰

3. **及时响应**
   - 5分钟内回复
   - 说到做到
   - 主动汇报进度

4. **情绪管理**
   - 保持耐心
   - 积极态度
   - 换位思考

### 禁忌话术

❌ "这个不行"  
✅ "这个有点难度，但我们可以试试..."

❌ "我不知道"  
✅ "这个我需要确认一下，马上答复您"

❌ "你要什么就说"  
✅ "为了更好服务您，能否告诉我..."

❌ "别家做不了"  
✅ "我们在这方面有独特优势..."

## ⚠️ 注意事项

### 沟通原则
1. **真诚第一：** 不夸大、不虚假承诺
2. **专业为本：** 展现专业知识和能力
3. **服务至上：** 站在客户角度思考
4. **效率优先：** 快速响应、高效执行

### 特殊情况处理
- **客户投诉：** 先道歉、快速响应、跟踪解决
- **价格争议：** 解释成本、强调价值、适度让步
- **交期延误：** 提前告知、诚恳道歉、补偿方案
