## 服务流程通用模板

```
需求沟通 → 方案策划 → 报价确认 → 设计/制作 → 交付验收
```

## 报价原则

1. **按项目报价** - 展览、活动、VI
2. **按月度报价** - 自媒体代运营
3. **设计+生产分开** - 包装设计
4. **套餐+单项** - 灵活组合
