## 二、服务流程

```
需求沟通 → 品牌调研 → 概念设计 → LOGO提案 →
深化设计 → VI手册 → 应用延展 → 交付源文件
```

| 阶段 | 周期 | 交付物 |
|------|------|--------|
| 品牌调研 | 3-5天 | 调研报告 |
| LOGO设计 | 5-10天 | 3-5个方案 |
| VI基础 | 5-7天 | 基础系统规范 |
| VI应用 | 7-15天 | 应用系统设计 |
| VI手册 | 3-5天 | 完整VI手册 |

## 六、常见问题

**Q: LOGO设计包含几个方案？**
A: 标准版5个方案，基础版3个方案。

**Q: 修改次数有限制吗？**
A: 通常3次大修改，小调整不限。

**Q: 可以只做LOGO不做VI吗？**
A: 可以，单独LOGO设计也接。

**Q: VI手册是什么格式？**
A: PDF手册+AI源文件，方便后续使用。
