| 服务 | 说明 |
|------|------|

| 类型 | 适用产品 |

### 1.3 一站式服务

```
需求沟通 → 市场调研 → 概念设计 → 结构设计 →
视觉设计 → 打样确认 → 批量生产 → 交付
```

## 二、服务流程

| 阶段 | 周期 | 交付物 |
|------|------|--------|
| 需求沟通 | 1天 | 设计brief |

## 四、案例场景

## 五、设计要点

### 5.1 包装设计原则

| 原则 | 说明 |
|------|------|
| 保护性 | 保护产品不受损 |
| 便利性 | 方便运输、开启 |
| 美观性 | 吸引消费者 |
| 环保性 | 材料可回收 |
| 经济性 | 成本可控 |

### 5.2 常见盒型

| 盒型 | 特点 | 适用 |
|------|------|------|
| 折叠盒 | 成本低、运输方便 | 常规产品 |
| 天地盖 | 高档、展示好 | 礼品 |
| 书型盒 | 仪式感强 | 高端礼品 |
| 抽屉盒 | 精致、有趣 | 首饰、化妆品 |
| 异形盒 | 独特、记忆点 | 创意产品 |
