## 工艺选择速查

### 表面效果

| 想要效果 | 推荐工艺 |
|---------|---------|
| 光亮鲜艳 | 覆亮膜 |
| 高级质感 | 覆哑膜 |
| 金属光泽 | 烫金/烫银 |
| 立体高光 | 局部UV |
| 凹凸触感 | 击凸/压纹 |
| 特殊形状 | 模切 |

### 装订选择

| 页数 | 预算 | 推荐装订 |
|------|------|---------|
| 8-64P | 低 | 骑马钉 |
| 20P+ | 中 | 无线胶装 |
| 40P+ | 高 | 锁线胶装 |
| 不限 | 高端 | 精装 |
| 不限 | 翻阅多 | 线圈装 |

### 喷绘选择

| 场景 | 推荐工艺 |
|------|---------|
| 室内近看 | 写真 |
| 户外远看 | 喷绘 |
| 硬质材料 | UV打印 |
| 立体字 | 雕刻 |
