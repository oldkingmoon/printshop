# 画册/书刊产品知识库

> 分类: 印刷品 > 商务印刷 > 画册/书刊
> 更新: 2026-02-06

## 一、产品概述

画册/书刊是企业宣传、产品展示的重要载体，包括企业画册、产品目录、期刊杂志、说明书等。

## 二、规格选项

### 2.1 尺寸

| 规格 | 尺寸 | 说明 |
|------|------|------|
| 16开 | 285×210mm | 最常用 ⭐ |
| 大16开 | 291×216mm | 略大 |
| A4 | 297×210mm | 国际标准 |
| 8开 | 420×285mm | 大画册 |
| A5 | 210×148mm | 小册子 |
| 正方形 | 210×210mm | 创意型 |

### 2.2 页数（P数）

| 页数 | 说明 |
|------|------|
| 8P | 最少（骑马钉） |
| 12P/16P | 小册子 |
| 20P/24P | 常规画册 |
| 32P/40P | 标准画册 ⭐ |
| 48P/64P | 厚画册 |
| 80P+ | 书刊/杂志 |

> 注：P = Page，1P = 1面，4P = 1张纸双面

### 2.3 封面纸张
