## 材料选择速查

### 印刷用纸

| 需求 | 推荐 |
|------|------|
| 色彩鲜艳 | 铜版纸 |
| 高级质感 | 哑粉纸 |
| 独特纹理 | 特种纸 |
| 挺括硬度 | 卡纸 |

### 喷绘材料

| 场景 | 推荐 |
|------|------|
| 室内海报 | PP纸 |
| 照片输出 | 相纸 |
| 灯箱 | 灯片/灯布 |
| 户外广告 | 户外背胶 |
| 车身广告 | 车贴 |

### 板材选择

| 场景 | 推荐 |
|------|------|
| 临时展示 | KT板 |
| 户外展板 | PVC板 |
| 立体造型 | 雪弗板 |
| 高端标牌 | 亚克力 |
| 户外耐久 | 铝塑板 |
